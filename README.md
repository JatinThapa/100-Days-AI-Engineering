# 🚀 100 Days of AI Engineering

Welcome to my **100 Days of AI Engineering** journey!

Over the next 100 days, I'll be learning, experimenting, and building with some of the most exciting technologies in modern AI, including:

🤖 Large Language Models (LLMs)  
📚 Retrieval-Augmented Generation (RAG)  
🧠 AI Agents  
🗄️ Vector Databases & Embeddings  
⚙️ Fine-Tuning  
🔍 Prompt Engineering  
🌐 AI-Powered Applications  

My goal isn't just to learn the theory behind these technologies—it's to understand how they can be combined with software engineering to build useful, intelligent applications.

Each day of this challenge will include:
- 📖 What I learned
- 🛠️ Projects and experiments
- 🤔 Challenges and debugging adventures
- 💡 Key takeaways
- 📈 Progress throughout the journey

From simple LLM integrations to full-fledged AI applications, this repository serves as a public log of my growth, successes, mistakes, and lessons learned along the way.
